Kurulum

LiMON AI'yı kurmak sadece birkaç dakika sürer. GitHub'dan indir, bağımlılıkları kur ve hemen kullanmaya başla.

  1. GitHub'dan LiMON AI reposunu indir
  2. Klasörü aç ve bağımlılıkları yükle
  3. LiMON AI'yı başlat ve ilk projeni oluştur
git clone https://github.com/aardaakpinar/limon.git
cd limon
npm install
npm run start
